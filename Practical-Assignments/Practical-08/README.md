# Practical Assignment 8: Decision Trees

This assignment involves building a **Decision Tree Classifier** using the **Pima Indians Diabetes Dataset** to predict the onset of diabetes.

## Objectives
1.  **Feature Selection:** Identifying Target vs. Feature variables.
2.  **Modeling:** Training a Decision Tree Classifier using `scikit-learn`.
3.  **Visualization:** Plotting the tree structure to interpret decision rules.
4.  **Mathematical Analysis:** Computing **Entropy**, **Information Gain**, and **Gini Index** to explain the selection of the root node.

## Libraries Used
* **Pandas:** Data handling.
* **Scikit-Learn:** Modeling (`DecisionTreeClassifier`) and plotting (`plot_tree`).
* **Matplotlib:** Visualizing the tree.
